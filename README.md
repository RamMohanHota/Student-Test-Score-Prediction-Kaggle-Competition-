# Student-Test-Score-Prediction-Kaggle-Competition-

🔹 Developed a high-performance machine learning model to predict student test scores using a large-scale dataset (~700K records) with complex, multi-dimensional features
🔹 Implemented advanced ensemble techniques, including a Bagging Meta Estimator, to improve model robustness and accuracy
🔹 ⚙️ Optimized model performance using LightGBM with GridSearchCV for effective hyperparameter tuning
🔹 🚀 Achieved the best results with LightGBM among all experimented models
🔹 📊 Generated a submission file with ~270K predictions for evaluation
🔹 🏆 Secured a rank of 1410 out of 7078 participants on the Kaggle platform
